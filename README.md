## 中文原书勘误
中文《C++ Primer》第五版包含了较多的错误，部分勘误如下：
### 1. P16 练习1.19
题目“修改你为1.4.1节练习`1.10`（第11页）...”改为“修改你为1.4.1节练习`1.11`（第11页）...”。
